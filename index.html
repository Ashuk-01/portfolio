How I AI

I design with AI, not against it.

At CleverX, I started using Claude Code not as an experiment — but as a real tool in production. I'd design a page, then ship it myself. Participant-facing screens, bug fixes, responsive layouts — all pushed directly through AI and Git, no handoff needed.

It changed how I think about speed. I started running UX audits faster — walking through dashboard pages, catching hierarchy and navigation issues, and shipping fixes the same day instead of filing tickets for the next sprint.

Now I use AI to prototype in hours, not days. Rough idea to working interface — fast enough to test with real feedback before committing to a direction. The thinking is still mine. The craft is still mine. AI just removed the waiting.

This portfolio — vibe coded with Claude Code and a lot of intention.
